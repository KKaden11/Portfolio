## Arrays

This blog is about arrays.

### Arrays Homework

Complete all exercises below.

### Exercise 1: Array Basics - Access Elements

```javascript
const favorites = ["Minecraft", "Interstellar", "Basketball", "Pizza", "Music"];

console.log("Entire array:", favorites);
console.log("First element:", favorites[0]);
console.log("Last element:", favorites[favorites.length - 1]);
console.log("Length:", favorites.length);
```

### Output (Solved)

```text
Entire array: [ 'Minecraft', 'Interstellar', 'Basketball', 'Pizza', 'Music' ]
First element: Minecraft
Last element: Music
Length: 5
```

### Exercise 2: Modify Arrays

```javascript
let shoppingList = ["milk", "eggs", "bread", "cheese"];

console.log("Original array:", shoppingList);

shoppingList[1] = "butter";
shoppingList.push("yogurt");
shoppingList.splice(shoppingList.indexOf("bread"), 1);

console.log("Final array:", shoppingList);
```

### Output (Solved)

```text
Original array: [ 'milk', 'eggs', 'bread', 'cheese' ]
Final array: [ 'milk', 'butter', 'cheese', 'yogurt' ]
```

### Exercise 3: Loop Through an Array

```javascript
const numbers = [10, 25, 30, 15, 20];
let sum = 0;

for (const num of numbers) {
  console.log("Number:", num);
}

for (const num of numbers) {
  console.log("Times 2:", num * 2);
}

for (const num of numbers) {
  sum += num;
}

console.log("Sum:", sum);
```

### Output (Solved)

```text
Number: 10
Number: 25
Number: 30
Number: 15
Number: 20
Times 2: 20
Times 2: 50
Times 2: 60
Times 2: 30
Times 2: 40
Sum: 100
```
