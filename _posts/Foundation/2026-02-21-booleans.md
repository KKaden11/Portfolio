## Booleans

Booleans are one of the most fundamental data types in programming.

- A boolean can only be `true` or `false`.
- Booleans are used to make decisions and control program flow.

### Basic Boolean Syntax

```javascript
let isStudent = true;
let isRaining = false;
```

```javascript
let isActive = true;
console.log(isActive);
console.log("Boolean value: " + isActive);
```

### Usage: Boolean Comparisons

You can create booleans using comparison operators:

```javascript
let age = 15;
let isAdult = age >= 18;
let isTeenager = age >= 13 && age <= 19;

console.log("Is adult: " + isAdult);       // false
console.log("Is teenager: " + isTeenager); // true
console.log(isAdult && isTeenager);        // false
console.log(isAdult || isTeenager);        // true
```

### Boolean Operators

Booleans can be combined with:

- `&&` (AND)
- `||` (OR)
- `!` (NOT)

### Solved Code: Challenge 1

```javascript
let hasLicense = true;
let hasCar = true;
let canDrive = hasLicense && hasCar;

console.log("Has license: " + hasLicense); // true
console.log("Has car: " + hasCar);         // true
console.log("Can drive: " + canDrive);     // true
```

### Solved Code: Challenge 2

```javascript
// 1) AND: number must be positive and even
function isPositiveAndEven(num) {
  let isPositive = num > 0;
  let isEven = num % 2 === 0;
  return isPositive && isEven;
}

// 2) OR: number is special if negative or even
function isSpecial(num) {
  let isNegative = num < 0;
  let isEven = num % 2 === 0;
  return isNegative || isEven;
}

// 3) NOT: opposite of positive
function isNotPositive(num) {
  return !(num > 0);
}

console.log("isPositiveAndEven(8):", isPositiveAndEven(8));   // true
console.log("isPositiveAndEven(7):", isPositiveAndEven(7));   // false
console.log("isSpecial(-3):", isSpecial(-3));                 // true
console.log("isSpecial(5):", isSpecial(5));                   // false
console.log("isNotPositive(5):", isNotPositive(5));           // false
console.log("isNotPositive(-1):", isNotPositive(-1));         // true
```
